# Country and Currency Exchange Insights

This application consists of a Node.js server and a simple web interface implemented using React.js. The application enables users to search for countries, view their details, and perform currency conversions.

## Installation

To install and run the application locally, follow these steps:

1. Clone the repository.
2. Navigate to the root directory.
3. Install the required dependencies using the package manager of your choice (npm or yarn).

```bash
npm install
```

4. Configure the environment variables, including the API keys for the third-party services.
5. Run the Node.js server and the React application concurrently.

```bash
npm run dev
```

6. Open your web browser and access the application at the provided address.

## Usage

To use the application, follow these steps:

1. Access the home view and search for countries.
2. Authenticate through the login view to obtain a valid JWT.
3. Use the web interface to search, add countries to the list, and perform currency conversions.

## APIs Used

- [Restcountries API](https://restcountries.com) - Used for country lookup and general information.
- [Fixer API](https://fixer.io) - Used for obtaining exchange rates.

## Technologies Used

- Node.js
- React.js
- Express
- MUI (Material-UI)
- Context API
- Prettier
